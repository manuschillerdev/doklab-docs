# Installation

import { Tabs } from 'nextra/components'
import { Callout } from 'nextra/components'

## Quick Install (Recommended)

One command installs doklab and all dependencies:

```bash
curl -fsSL https://get.doklab.dev | sudo bash
```

This installs:
- **doklab** - The Docker plugin
- **Docker** - Container runtime (if not present)
- **sops** - Secret encryption
- **vals** - Secret resolution
- **kopia** - Backup management

<Callout type="info">
  Supports Debian, Ubuntu, and derivatives. For other systems, see manual installation below.
</Callout>

## Verify Installation

```bash
docker lab version
docker lab doctor
```

`doctor` checks all dependencies and shows their status.

## Initialize

<Tabs items={['Tailscale (Private)', 'Cloudflare (Public)']}>
  <Tabs.Tab>
    For private access via Tailscale mesh network:

    ```bash
    docker lab init --domain example.ts.net --ts-authkey tskey-auth-xxxxx
    ```

    Get your auth key from the [Tailscale admin console](https://login.tailscale.com/admin/settings/keys).
  </Tabs.Tab>
  <Tabs.Tab>
    For public access via Cloudflare:

    ```bash
    docker lab init --cloudflare --domain example.com --cf-token YOUR_TOKEN --acme-email you@example.com
    ```

    Create a token with Zone:DNS:Edit, Zone:Zone:Read, and Account:Cloudflare Tunnel:Edit permissions.
  </Tabs.Tab>
</Tabs>

That's it. You're ready to deploy.

## Manual Installation

<Tabs items={['apt (Debian/Ubuntu)', 'macOS', 'Binary']}>
  <Tabs.Tab>
    ```bash
    # Add repository
    curl -fsSL https://apt.doklab.dev/gpg | sudo gpg --dearmor -o /usr/share/keyrings/doklab.gpg
    echo "deb [signed-by=/usr/share/keyrings/doklab.gpg] https://apt.doklab.dev stable main" | sudo tee /etc/apt/sources.list.d/doklab.list

    # Install
    sudo apt update
    sudo apt install doklab
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    brew tap manuschillerdev/doklab
    brew install doklab
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    curl -L https://github.com/manuschillerdev/doklab/releases/latest/download/doklab-linux-amd64 -o /usr/local/bin/doklab
    chmod +x /usr/local/bin/doklab
    ```

    <Callout type="warning">
      Binary install requires manual dependency installation (Docker, sops, vals, kopia).
    </Callout>
  </Tabs.Tab>
</Tabs>
